## 🎨 Design System

### Color Palette
- **Primary:** Burgundy / Blood Red (#b02700, #dc3300)
- **Secondary:** Deep Blue (#005fad)
- **Tertiary:** Muted Teal (#4d5f61)
- **Surface:** Clean neutrals with high contrast

### Typography
- **Headlines:** Epilogue (800-900 weight)
- **Body:** Inter
- **Labels:** Space Grotesk

### Features
- 🌓 Light/Dark mode support
- 🎯 Material Design 3 inspired components
- ⚡ Smooth animations with GSAP
- 📱 Fully responsive
- 🎬 Cinematic hero section

---

## 🛠 Tech Stack

- **Build Tool:** Vite 8.x
- **Styling:** Tailwind CSS 4.x
- **Animations:** GSAP 3.x
- **Icons:** Material Symbols

---

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
